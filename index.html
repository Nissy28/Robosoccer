Firebase real-time sync

To sync scores across all iPads in real time, connect a free Firebase Realtime Database. All referees on the same Wi-Fi (or internet) will see score updates instantly.

How to get your Firebase URL:
1. Go to console.firebase.google.com
2. Create a new project → Realtime Database → Create database (test mode)
3. Copy the URL shown (ends in .firebaseio.com) and paste above
4. Open this same HTML file on all iPads and enter the same URL

Clears all teams, matches, and scores from this device (and Firebase if connected).
